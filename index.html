About BBS Door Games

Before the World Wide Web, computer enthusiasts connected to Bulletin Board Systems (BBSs) using dial-up modems. "Door games" were external programs that users could launch from the BBS menu, offering everything from combat adventures to trading simulations.

This collection recreates that nostalgic experience with faithful Python implementations of classic door games, now playable in your web browser!
